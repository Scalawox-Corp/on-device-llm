## ASUS Ascent GX10 AI 슈퍼컴퓨터를 위한 [스칼라웍스](https://www.scalawox.com/)의 On Device형 LLM 패키지


|패키지|웹사이트|설명|
|:---|:---|:---|
|LM Studio|https://lmstudio.ai/|*gpt-oss, Qwen3, Gemma3, DeepSeek* 등의 `로컬 LLM`을 데스크탑 컴퓨터에서 **쉽고 안전**하게 실행하는 백엔드 애플리케이션|
|Open WebUI|https://openwebui.com/|`웹 UI` 기반의 자체 호스팅 AI 플랫폼으로, 완전히 <u>오프라인</u>에서 작동하고 다양한 LLM 백엔드 애플리케이션 및 OpenAI 호환 API를 지원하며, <u>RAG용 추론 엔진</u>이 내장된 강력한 AI 배포 솔루션|
|Visual Studio Code|https://code.visualstudio.com/|*세계에서 가장 사용되는* 오픈소스 `AI 코드 에디터`|
|cline|https://cline.bot/|*Visual Studio Code*를 위한 `코딩 AI 에이전트`|
